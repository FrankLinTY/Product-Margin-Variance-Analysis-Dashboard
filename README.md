This Product Margin Rate Variance Dashboard decomposes period-over-period weighted-average margin changes into Mix, Volume, and Rate effects. Built as a standalone HTML file, it features embedded CSS styling following a clean corporate design and JavaScript for dynamic client-side rendering. The application visualizes interactive key performance indicators, a product-by-factor impact matrix, an absolute impact driver treemap, and a sortable tabular drill-down using synthetic portfolio data. It serves as a reusable, domain-agnostic analytics framework to help teams instantly identify business drivers, margin compression, or volume shifts across product segments without complex server dependencies.
